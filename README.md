# Java Calculator

A simple Java-based calculator application that performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

---

##  Features
- Addition  
- Subtraction  
-  Multiplication  
-  Division  
- User-friendly input using console  
- Fast and efficient calculations  

---

## Technologies Used
- Java  
- OOP Concepts  
- Scanner class for input  

---

## How to Run

1. Open the project in any Java IDE (Eclipse / IntelliJ / VS Code)  
2. Compile the program  
3. Run the file  
4. Enter two numbers  
5. Choose operation (+, -, *, /)  
6. Get the result  

---

## Concepts Used
- Conditional statements (switch-case)  
- User input handling  
- Basic arithmetic logic  

---

## Example
Enter two numbers:
10
5
Choose operation: + - * /
+
Result: 15
